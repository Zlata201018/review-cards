Review cards:

How implement more visual activity and interest to a web page?

🎓 With the use of the transitions and animations proporties.

Which property specifies the name of the @keyframes animation to apply?

🎓 animation-name

Which property defines how long an animation cycle takes to complete?

🎓 animation-duration

Which property sets the speed of the animation?

🎓 animation-timing-function

Which property sets a delay before the animation starts?

🎓 animation-delay

Which property specifies how many times the animation should repeat?

🎓 animation-iteration-count

Which property is a shorthand for setting all transition properties in one declaration?

🎓 transition

Which property specifies when the transition effect will start?

🎓 transition-delay
